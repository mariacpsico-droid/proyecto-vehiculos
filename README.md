# Aplicación Web de Análisis de Vehículos

Esta aplicación web interactiva permite explorar un conjunto de datos de anuncios de venta de coches mediante visualizaciones dinámicas.

## Funcionalidades

- **Histograma interactivo**: Visualiza la distribución de diferentes variables del dataset
- **Gráfico de dispersión**: Explora las relaciones entre variables numéricas
- **Interfaz intuitiva**: Controles fáciles de usar para generar visualizaciones

## Tecnologías utilizadas

- Python
- Streamlit
- Plotly Express
- Pandas

## Cómo ejecutar la aplicación

1. Instala las dependencias: `pip install -r requirements.txt`
2. Ejecuta la aplicación: `streamlit run app.py`
